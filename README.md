 Farmer & Consumer App

About

A simple web app that connects **farmers** with **consumers** directly.

Farmers can add products and consumers can view them.

---

 Technology Used

* Python (Flask)
* SQLite
* HTML, CSS, JS



 Features

* Farmer login/signup
* Consumer login/signup
* Add products
* View products

---

 Run Project


git clone https://github.com/your-username/your-repo.git
cd your-repo
pip install -r requirements.txt
python app.py
